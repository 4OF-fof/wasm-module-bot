# Agent Instructions

- Run pnpm-related commands outside the sandbox.
- Do not start development servers or long-running server processes, including commands such as `pnpm dev`, `pnpm run dev`, `pnpm start`, or similar server launch commands.
- Do not run install commands such as `pnpm install` without explicit user approval.
- If a pnpm command is required for verification or maintenance, explain the reason and request the appropriate approval before running it.
